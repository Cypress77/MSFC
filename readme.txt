Описание:

Предупреждение!!! Статистика не будет работь без базы MySQL. Если у вас нет возможности использовать базу MySQL, скачайте последнюю версию модуля первой версии.

Необходимо что бы были установлены следующие компоненты:

php-curl
php-pdo
php-pdo_mysql
